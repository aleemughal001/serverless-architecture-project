# AWS Lambda REST API with External HTTP Request

## 📌 Project Overview

This project demonstrates how to build and deploy an AWS Lambda function that makes an external HTTP request using the `requests` library, exposed through Amazon API Gateway.

The Lambda function fetches data from a public API and returns it as a JSON response.

---

## 🏗 Architecture

Client (Postman)  
→ API Gateway  
→ AWS Lambda  
→ External API (jsonplaceholder.typicode.com)

---

## 🛠 Technologies Used

- AWS Lambda
- Amazon API Gateway
- Python 3.x
- AWS Lambda Layers
- Requests library
- Postman (API testing)
